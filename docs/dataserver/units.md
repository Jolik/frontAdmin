## 4.1.Список единиц измерений

**Назначение:** Получить полный список единиц измерений, что зарегистрированы в сервере

**Запрос** `/api/v2/units/list`

**Тип запроса:** GET

**Возможные параметры**:

| **имя поля** | **тип** | **описание**                                                              |
|--------------|---------|---------------------------------------------------------------------------|
| `page  `     |         | текущая страница для отображения                                          |
| `pagesize`   |         | кол-во элементов на странице для отображения. По умолчанию pagesize = 100 |
| `dest`       |         | выдавать только те измерения, которые конвертируются в указанную          |
| `flag`       |         |                                                                           |
| `-formula`   |         | выдавать единицы измерений вместе с формулами                             |

**Пример:**  
`/api/v2/units/list?dest=1-02-01`
`/api/v2/units/list?page=2`

**Возможные поля единиц измерений**:

| **имя поля** | **тип** | **описание**                                                        |
|--------------|---------|---------------------------------------------------------------------|
| `page`       | string  | текущий номер страницы (дублирует параметр запроса)                 |
| `pagecount`  | string  | общее число страниц (зависит от размера pagesize)                   |
| `pagesize`   | string  | сколько элеметнов выдавать на странице (дублирует параметр запроса) |
| `total`      | string  | всего элементов в таблице                                           |
| `uid `       | string  | уникальный идентификатор единицы измерения                          |
| `mid`        | uuid    | идентификатор метадаенных связанный с единице измерения             |
| `name `      | string  | название измерения                                                  |
| `caption  `  | string  | символ для отображения                                              |
| `convert`    | []json  | список возможных конверсий опционально                              |
